 Las variables son una forma que el lenguaje nos da para guardar información que querramos usar en otro momento,
para que sea fácil de encontrar (y modificar). Podemos dividirla en dos partes principales. La declaración y el tipo.  
 La declaración indica que tienen una forma particular de escribirse que nos dá información:
 La sintaxis empieza con la palabra clave "var" que le avisa a la máquina que lo que le estamos pasando es
una variable.
 Luego viene el NOMBRE de la variable (arbitrario). Que es la referencia que nos sirve para encontrarla y nos
ayuda a recordar algo sobre el valor guardado (qué importancia tiene).

  El símbolo asignación " = " va a vincular la variable declarada (palabraclave + nombre) con el valor que tiene.

 El VALOR de una variable es su contenido o estado. Lo que al fin de cuentas se quería guardar para ser usado después.

 El TIPO clasifica a las variables por lo que contienen (sea un número, una cadena de texto, etc); y establece las
 operaciones que se podrán realizar con el contenido de una variable.
 
  Hay lenguajes en el que el tipo de la variable tiene que estar declarado junto con esta. En JS no es necesario.

 Una característica importante de las variables es que pueden ir cambiando de valor durante la ejecución de un código,
pero también se pueden declarar constantes para que el valor que guarda no se modifique. 

Los STRINGS son caracteres encadenados se pueden utilizar como un solo elemento con distintas cualidades. Tambien se
pueden realizar cambios en estas cualidades pero fundamentalmente no se podrán realizar operaciones matemáticas al mismo.
